Проект не связан, не аффилирован и никак не поддерживается компанией "Powernet" (ООО "Пауэрнет"). Использовать программу вы можете исключительно на свой страх и риск. В данном публичном репозитории **отсутствуют** секретные ключи, дампы баз данных или закрытые модули API провайдера.

---

## Основной функционал

*   **Bento Dashboard** — единый пульт управления всеми вашими договорами на одном экране.
*   **Searchable Cam Matrix** — интерактивное выпадающее меню с личными камерами и домофонами, дополненное мгновенным поиском и ограничением по высоте.
*   **iMessage CommLink** — полноценный чат с технической поддержкой в реальном времени, работающий напрямую через WebSockets (`wss://`) по оригинальному протоколу провайдера.
*   **Dynamic Island Toast** — мягкие всплывающие уведомления в верхней части экрана (в стиле iOS Dynamic Island) вместо стандартных системных алертов.
*   **Trust Override** — система автоматически выведет кнопку активации доверительного платежа прямо на карточку объекта, если баланс упадет ниже критического уровня.
*   **Instant SBP Pay** — генерация СБП QR-кодов прямо на экране компьютера для моментального пополнения баланса со смартфона.
*   **Solid Security** — все конфиденциальные данные (логины/пароли) шифруются и хранятся локально в системных директориях через модуль `electron-store`.

## 📁 Структура репозитория (Repository Tree)

Для предотвращения DMCA-страйков и сохранения безопасности сетевая логика взаимодействия с серверами провайдера изолирована в приватный модуль:

```text
POWERNET_OS/
├── private_modules/             # Секретная папка (добавлена в .gitignore)
│   ├── powernet-api/
│   │   └── index.js             # Логика работы с API
│   └── powernet_map_dump.json   # Локальный дамп городских камер
├── index.html                   # Оболочка UI (Tailwind, Lucide, Hls.js)
├── main.js                      # Главный процесс Electron (Прокси заголовков)
├── preload.js                   # Безопасный мост (Context Bridge)
├── package.json                 # Зависимости и скрипты сборки
└── .gitignore                   # Защита от случайного слива приватных папок
